Recommended repos & libraries
Curated specifically for Tanmatra's stack (React 19 + Vite + Hono + tRPC + Drizzle/MySQL + shadcn/ui + Tailwind) and its domain (food delivery, rider tracking, AI ops agent, hospital flavor).

Top 5 picks (highest leverage)
vercel/ai — https://github.com/vercel/ai — rewrite the support agent on this; biggest immediate win
satnaing/shadcn-admin — https://github.com/satnaing/shadcn-admin — copy layouts for /admin/ops
medusajs/nextjs-starter-medusa — https://github.com/medusajs/nextjs-starter-medusa — steal the cart + checkout flow (yours is currently faked: cartCount={3})
visgl/react-map-gl + socketio/socket.io — make /track actually live with rider markers + websocket push
taskforcesh/bullmq — https://github.com/taskforcesh/bullmq — kill the setTimeout foot-gun in order-pipeline.service.ts before it bites in production

AI ops agent (Tanmatra's standout feature)
Your support agent currently regex-parses Gemini responses. Strongly recommend migrating to one of:

Repo	Why
vercel/ai — https://github.com/vercel/ai	The AI SDK — first-class Gemini support, native tool calling, streaming, useChat hook. Replaces ~80% of support-agent.service.ts.
vercel/ai-chatbot — https://github.com/vercel/ai-chatbot	Reference UI for the floating chat widget.
google-gemini/cookbook — https://github.com/google-gemini/cookbook	Gemini 2.5 function-calling examples.
Yonom/assistant-ui — https://github.com/Yonom/assistant-ui	Pre-built React chat UI on top of the AI SDK.
langchain-ai/langgraph — https://github.com/langchain-ai/langgraph	State-machine agents with the human-in-the-loop pattern your code already gestures at (needsConfirm, approveAction).
Background jobs (fix the setTimeout race)
Repo	Why
taskforcesh/bullmq — https://github.com/taskforcesh/bullmq	Redis-backed jobs. Industry standard. Ideal for delayed rider-assignment + retries.
graphile/worker — https://github.com/graphile/worker	If you ever migrate to Postgres — uses the DB itself, no Redis needed.
triggerdotdev/trigger.dev — https://github.com/triggerdotdev/trigger.dev	Managed if you'd rather not run infra.
Auth alternatives (if you ever leave Kimi)
Repo	Why
better-auth/better-auth — https://github.com/better-auth/better-auth	Modern, framework-agnostic, works great with Drizzle + Hono. Best replacement for your hand-rolled jose JWT setup.
lucia-auth/lucia — https://github.com/lucia-auth/lucia	In maintenance mode but the docs remain the best learning resource for session auth.
